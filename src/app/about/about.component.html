The USA Pickleball Player Skill Ratings rubric provides specific skills to achieve each level of player rating. The rubbric is distributed in PDF file format. This app digitizes the PDF rubric to make it easy for Pickleball players to complete skill assessment from any where and on any devices (desktop, tablet or mobile).

Product Roadmap

  • Phase 1 - Online Skill Assessment form for skill level 2 to 5
  • Phase 2 - Enable login using Facebook or Google account and save skill assessment data
  • Phase 3 - Establish skill matrix and link to training resources
