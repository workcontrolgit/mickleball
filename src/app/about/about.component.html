The USA Pickleball Player Skill Ratings rubric provides specific skills to achieve each level of player rating. The rubric is distributed in read-only PDF file format. The Mickleball app transform the rubric to fillable form and makes it easy for players and coaches to complete skill assessment from any where and on any devices (desktop, tablet, or mobile).

The rating levels are progressive, ranging from 2.0 to 5.5. The higher level demands more advanced skills. A set of skills are defined and associated to each rating level. In Mickleball app, each skill can be individually evaluated using academic grading scale ABCD. The rating result is shown in user-friendly Report Card format.
